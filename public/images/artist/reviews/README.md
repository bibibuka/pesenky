# Concert Reviews / Отзывы о концертах

## Как добавить отзыв

Просто закинь картинки (скриншоты отзывов) в эту папку — они автоматически появятся на странице концертов.

## Поддерживаемые форматы
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`
- `.svg`
- `.heic` / `.heif`

## Порядок отображения
Картинки сортируются по имени файла (алфавитно-цифровой порядок).
Если нужен определённый порядок — называй файлы с номерами:
`01.jpg`, `02.jpg`, `03.jpg` и т.д.

## Дубликаты оригиналов
Если нужны оригиналы без сжатия — клади их в:
`public/images-original/artist/reviews/`
