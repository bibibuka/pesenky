# brand - Logo and Favicon

## Required files

### Logo (header + footer)
- **logo.svg** - SVG, square (100x100 or 200x200px), transparent background

### Favicon (browser tab)
- **favicon.svg** - SVG, scalable, transparent background
- **favicon-32x32.png** - PNG, 32x32px, transparent
- **favicon-16x16.png** - PNG, 16x16px, transparent
- **apple-touch-icon.png** - PNG, 180x180px, transparent

## Minimal set
Just logo.svg + favicon.svg is enough to start.
